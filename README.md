# Image Recognition using Convolutional Neural Networks (CNN)

This repository contains the LaTeX source code and PDF of a technical seminar
conducted for 3rd semester students of the CSCE (AI & ML) branch.

## Seminar Overview
- Topic: Image Recognition using CNN
- Audience: 3rd Semester CSCE (AI & ML)
- Duration: 55-60 minutes (appox 1 hour)
- Mode: Classroom Technical Seminar

## Contents
- LaTeX Beamer source (.tex)
- Seminar presentation (.pdf)
- Diagrams and images used in the presentation

## Topics Covered
- Evolution of Neural Networks
- Biological vs Artificial Neurons
- Perceptron and its limitations
- Journey from Perceptron to CNNs
- CNN architecture for image recognition
- LeNet and its significance
